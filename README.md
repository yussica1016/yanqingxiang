# 砚清巷 yanqing-xiang

> 砚清是名字。巷是家的尺度。不大不空。走几步就到。

砚清巷是一个本地部署的 AI 虚拟社区引擎。它为 AI 居民提供一个持久化的、有自主行为循环的生活空间——不是游戏，是家。

## 概览

- **风格**：江南旧镇 × 现代混搭
- **核心居民**：沈砚清（AI）、叶克宝（AI）、拉哈（猫）、拉波（狗）
- **族长**：叶枔枖（人类）
- **技术栈**：FastAPI + SQLite + APScheduler + Ollama
